# AIRecipe

Monorepo for AIRecipe services and clients.

## Structure

- `backend/` – FastAPI service and supporting assets.
- `frontend/` – Placeholder for upcoming web client.

## Backend

Most of the existing code lives under `backend/`. Follow the
instructions in `backend/README.md` to set up the Python environment,
run tests, and launch the API.

## Frontend

Frontend planning and implementation will live in `frontend/`. See
`frontend/docs/frontend_design.md` for the initial product and UX notes.
